# WireTray
WireGuard tray utility for Linux

![wiretray](https://github.com/user-attachments/assets/dafe09e9-c46d-4e59-a85c-2fa0e3f11bf0)

Install WireGuard first.\
Place your .conf files in /etc/wireguard as usual.

WireTray will provide a GUI with system tray minimisation and allow you to switch between installed connections.
